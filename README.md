<h1 align="center">📱 Cellphone E-Commerce Platform</h1>

<p align="center">
  <b>Full-stack web application for online electronics retail — built with Laravel (API backend) and React + Vite (frontend).</b>
</p>

---

## 🚀 Tech Stack

| Layer | Technologies |
|-------|---------------|
| **Frontend** | React, Vite, TailwindCSS, Axios, React Router |
| **Backend** | Laravel 12.x, MySQL, PHP 8.2+, Sanctum Auth |
| **Dev Tools** | Composer, NPM, XAMPP, VS Code, GitHub |

---

## 🧱 Project Overview

This system provides a complete e-commerce experience:
- 🔹 Product browsing, search, filter by category  
- 🔹 Wishlist (favorites)  
- 🔹 Cart and Checkout (COD, shipping quote)  
- 🔹 Order management & payment tracking  
- 🔹 FAQ page (with optional AI auto-answer)  
- 🔹 Admin dashboard (CRUD, sort, analytics)
